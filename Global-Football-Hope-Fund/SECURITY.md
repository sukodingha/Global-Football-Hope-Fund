# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Global Football Hope Fund, please report it responsibly.

Please do not publicly disclose the issue until it has been reviewed and addressed.

### What to Include

- a clear description of the vulnerability
- steps to reproduce the issue
- potential impact
- any relevant screenshots or logs

### How to Report

Please send your report to the project maintainer through a private channel or direct contact.

## Response Expectations

- acknowledgment within a reasonable time
- investigation and assessment of the report
- updates as the issue is addressed

Thank you for helping keep the project secure.
